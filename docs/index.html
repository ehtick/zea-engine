Parameters
ctx (CanvasRenderingContext2D)
x (Number) The top left x coordinate
y (Number) The top left y coordinate
width (Number) The width of the rectangle
height (Number) The height of the rectangle
radius (Number = 5) The corner radius; It can also be an object to specify different radii for corners
Name Description
radius.tl Number (default 0) Top left
radius.tr Number (default 0) Top right
radius.br Number (default 0) Bottom right
