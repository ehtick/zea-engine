Class: Quat

Quat

Class representing a quaternion. Quaternions are used to represent rotations without encountering gimble lock. Based on complex numbers that are not easy to understand intuitively.

new Quat(x, y, z, w)

Create a quaternion.
Parameters:
Name Type Default Description
x number 0 The angle of the x axis. Default is 0.
y number 0 The angle of the y axis. Default is 0.
z number 0 The angle of the z axis. Default is 0.
w number 1 The w value. Default is 1.
